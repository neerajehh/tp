I'm your CEG course planning assistant!
Type 'help' to see available commands.
List of commands:
Command: view grad req
- Print CEG graduation requirements
Command: prereq <module code>
- Show prerequisites for a module
Command: module <module code>
- Show information for a module
Command: available <module code> <semester>
- Check if module is available in a specific semester
Command: specialisations
- List all CEG specialisations
Command: specialisation <name>
- Show detailed info for a specialisation
Command: bye
- Exit ClassMate
